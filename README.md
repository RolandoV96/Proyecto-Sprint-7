# Proyecto-Sprint-7

esta aplicacion funciona para mostrarnos una idea de la relacion entre el costo de un auto usado y el kilometraje que trae.

primero lo que realiza es un corte de los datos para que dependiendo su nivel de odometro lo encasille en y nos permita hacer un histograma mas representativo

seguido tenemos los botones, el primero es para crear un diagrama de dispersion que muestra la relacion de precio del vehiculo junt con su nivel de odometro seguido tenemos el boton para hacer el histograma que nos muestra el conteo de los vehiculos y nos los categoriza por su nivel de odometro por medio de colores.
